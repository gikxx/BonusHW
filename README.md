# Исследование NER: Bonus Homework

**Цель**: Сравнить разные методы распознавания сущностей на новостях о Brexit.

**Структура**:
- `brexit_ner_analysis.ipynb` — полный код анализа.
- `all_predictions.csv` — предсказания трёх ML-моделей (LogisticRegression, MultinomialNB, LinearSVC).
- `deepseek/` — промпты и ответы модели DeepSeek.


**Ключевой вывод**: Контекстное понимание LLM даёт значительное преимущество в задачах NER, но традиционные ML-модели остаются актуальны там, где важны скорость и интерпретируемость.
